w3-system

Wild Wolf War System はビジュアルノベル向けの汎用エンジンです。

使用しているライブラリ

  • SDL(Simple Directmedia Layer)2
  • Win32 API
  • Microsoft .Net Framework 4.5

エンジンとしての完成度

  • エンジンコアシステム
    • エンジンログウィンドウ
    • SDL2組み込み
    • メニュー・基本機能
  • スクリプトサブシステム
  • レンダリングサブシステム
  • コントロールサブシステム
