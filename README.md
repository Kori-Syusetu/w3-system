# w3-system
***Wild Wolf War System*** はビジュアルノベル向けの**汎用**エンジンです。
## 使用しているライブラリ
- SDL(Simple Directmedia Layer)2
- Win32 API
- Microsoft .Net Framework 4.5
## エンジンとしての完成度
- [ ] エンジンコアシステム
- - [x] エンジンログウィンドウ
- - [x] SDL2組み込み
- - [ ] メニュー・基本機能
- [ ] スクリプトサブシステム
- [ ] レンダリングサブシステム
- [ ] コントロールサブシステム
